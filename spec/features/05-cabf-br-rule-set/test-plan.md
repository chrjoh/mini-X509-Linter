# Test Plan: CA/Browser Forum BR Rule Set

## Scope

Verify the four BR lints (`validity_max_398_days`, `cn_in_san`,
`no_internal_names_or_reserved_ip`, `ext_key_usage_server_auth_present`), the SAN/EKU
accessors, and the reserved-IP/internal-name classifier — all scoped to TLS leaf certs.

## Conventions

Per `.claude/rules/rust-testing-core.md`: SIFER, `.unwrap()`/`.unwrap_err()`, behaviour-
focused tests grouped per lint.

## Fixtures (`testdata/`)

- `cabf_br_validity_400_days.pem`, `cabf_br_cn_not_in_san.pem`, `cabf_br_internal_san.pem`,
  `cabf_br_missing_serverauth.pem`.
- `good.pem` is a compliant TLS leaf (CN in SAN, public name, serverAuth, ≤398 days).
- Regenerated by `testdata/generate.sh`.

## Unit Tests (in `reserved.rs`, owned by feature task 01)

- `is_reserved_ip`: true for 10.0.0.1, 127.0.0.1, 169.254.x, ::1, fe80::, 192.0.2.x;
  false for a public IP.
- `is_internal_name`: true for `foo.internal`, `host.local`, single-label `localhost`;
  false for `example.com`.

## Integration Tests (`crates/linter/tests/cabf_br.rs`)

- Per lint: fixture flagged with descriptive message; `good.pem` passes.
- `cn_in_san`: a CN absent from SAN is flagged (message names the CN).
- `no_internal_names_or_reserved_ip`: multiple bad SAN entries → multiple findings.
- `validity_max_398_days`: 400-day cert flagged (message names duration); ≤398 passes.
- `ext_key_usage_server_auth_present`: missing serverAuth flagged; present passes.
- All four BR lints `NotApplicable` on a CA cert.

## Edge Cases

- Mixed SAN: one public + one internal name → only the internal entry flagged.
- Validity exactly 398 days → passes; 399 → flagged (boundary).
- IPv6 reserved ranges classified correctly.
- CN matching SAN case-insensitively for dNSName (document chosen policy and test it).

## Verification Commands

```
cargo test
cargo clippy --all-targets -- -D warnings
cargo fmt --check
bash testdata/generate.sh
```

## Exit Criteria

All four BR lints and the reserved classifiers validated; leaf-only scoping confirmed;
verification commands pass.
